Software

Tools & Code

All software developed in the lab is open-source and available on GitHub. We welcome contributions and bug reports.

⚙️ Software

SpliceScan

Deep learning pipeline for detecting cryptic and aberrant splice sites in long-read (PacBio / ONT) RNA-seq data. Achieves state-of-the-art accuracy on benchmark datasets.

Python · v2.1.0 GitHub →
⚙️ Software

scRNAkit

A scalable, containerized Nextflow workflow for end-to-end single-cell RNA-seq analysis: alignment, QC, clustering, annotation, and trajectory inference.

Nextflow / Python · v1.4.2 GitHub →
⚙️ Software

eCLIP-Tools

Bioinformatics pipeline for processing eCLIP-seq data: adapter trimming, alignment, peak calling, motif enrichment, and visualization. Optimized for low-input samples.

Snakemake · v3.0.1 GitHub →
⚙️ Software

m6AMapper

Probabilistic framework for calling m6A sites from MeRIP-seq and direct RNA sequencing data with accurate site-level quantification and differential methylation testing.

R package · v1.2.0 GitHub →
⚙️ Software

RBPnet

Graph neural network model for predicting RNA–protein binding affinities from sequence and secondary structure. Pre-trained on 150 eCLIP-seq datasets.

Python / PyTorch · v0.9.0 GitHub →
⚙️ Software

MultiOmicsKit

Framework for integrating RNA-seq, ATAC-seq, and proteomics data using factor analysis and regularized multi-block PCA methods. Includes interactive visualization.

R / Python · v1.0.0 GitHub →
Data

Published Datasets

Raw and processed sequencing data are deposited in GEO / SRA. Processed files and metadata tables are also available via Zenodo.

📊 Dataset

Single-cell splicing atlas — human neuronal differentiation

scRNA-seq + long-read data from 14 time points of iPSC-to-neuron differentiation. 42,000 cells, 15 donors. GEO: GSE000001.

Nature Neuroscience, 2025 GEO →
📊 Dataset

ALS motor neuron eCLIP-seq compendium

eCLIP-seq profiles for 28 RBPs in control and ALS patient-derived motor neurons. Includes processed peak files and binding site annotations. GEO: GSE000002.

Cell Reports, 2024 GEO →
📊 Dataset

Glioblastoma lncRNA CRISPR screen

Genome-wide CRISPR interference (CRISPRi) screen targeting 5,800 lncRNA loci in U87-MG cells. Raw counts and gene-level fitness scores. GEO: GSE000003.

Cancer Cell, 2023 GEO →
📊 Dataset

m6A atlas across 12 human cell lines

MeRIP-seq and m6A-seq under basal and stress conditions across 12 cell lines. Over 200,000 m6A sites with quantified methylation levels. Zenodo: 0000001.

Molecular Cell, 2025 Zenodo →
Protocols

Lab Protocols

Detailed experimental protocols developed and optimized in our lab. All protocols are hosted on protocols.io and are freely accessible.

🔬 Protocol

Low-input eCLIP-seq (10,000 cells)

Optimized eCLIP-seq protocol for rare cell populations and low-abundance RBPs. Includes adapter ligation, size-selection, and library prep steps.

Nature Protocols, 2023 protocols.io →
🔬 Protocol

RNA interactome capture (RIC) from mammalian cells

Poly(A)-capture-based method to isolate all RNA-binding proteins from total cell lysate. Optimized for HEK293T, U87-MG, iPSC-derived neurons.

Updated 2024 protocols.io →
🔬 Protocol

CRISPRi screen for lncRNA function

Step-by-step protocol for genome-scale CRISPRi screens targeting lncRNA loci, including sgRNA library design, lentiviral delivery, and NGS readout.

Updated 2023 protocols.io →
🔬 Protocol

smFISH for lncRNA localization

Single-molecule FISH protocol optimized for detecting low-abundance lncRNA transcripts in cultured cells and cryosectioned tissue. Probe design guidelines included.

Updated 2025 protocols.io →
